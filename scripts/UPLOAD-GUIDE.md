## 上传方式

### 方式1：拖拽上传（最简单）

1. 在IDE中打开文件浏览器
2. 导航到 `assets` 文件夹
3. 从你的电脑直接拖拽34张图片文件到IDE窗口
4. 等待上传完成

### 方式2：使用上传按钮

1. 在IDE中点击 `assets` 文件夹
2. 点击右上角的"上传"或"+"按钮
3. 选择你本地保存的34张图片文件
4. 点击确认上传

### 方式3：复制粘贴

1. 在你的电脑上选中34张图片文件
2. 按 Ctrl+C (Windows) 或 Cmd+C (Mac) 复制
3. 在IDE中打开 `assets` 文件夹
4. 按 Ctrl+V (Windows) 或 Cmd+V (Mac) 粘贴
5. 等待上传完成
